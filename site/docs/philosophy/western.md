# 🏛️ 西方哲学

西方哲学从古希腊开始，经历了数千年的发展和演变。

## 📖 历史发展阶段

### 古代哲学（公元前6世纪-公元2世纪）

#### 主要流派

**苏格拉底前哲学家**
- 泰勒斯：水是万物的本源
- 毕达哥拉斯：数学和谐论

**古典时期**
- 苏格拉底：知识即德行
- 柏拉图：理论世界论
- 亚里士多德：实体论和逻辑学

### 中世纪哲学（5-15世纪）

- 奥古斯丁：信仰与理性的结合
- 托马斯·阿奎那：自然神学
- 唯名论 vs 唯实论的争论

### 启蒙运动（17-18世纪）

| 哲学家 | 主要思想 |
|--------|---------|
| 笛卡尔 | 理性主义、"我思故我在" |
| 洛克 | 经验主义、社会契约论 |
| 康德 | 批判哲学、先验观念论 |
| 卢梭 | 自然状态论、共和制思想 |

### 现代哲学（19-20世纪）

!!! note "现代哲学的特点"
    强调理性的批判，关注人的主体性，反思科学和人文的关系。

#### 主要流派

- **德国观念论**：黑格尔的辩证法
- **功利主义**：边沁、穆勒
- **存在主义**：海德格尔、萨特
- **后现代主义**：德里达、福柯

## 🔑 核心概念

### 存在论

??? example "什么是存在？"
    存在论研究存在的本质和性质。从不同的哲学传统看：
    
    - **本质主义**：存在有固定的本质
    - **存在主义**：存在先于本质
    - **现象学**：通过现象研究存在的结构

### 认识论

关于人类知识的来源、本质和有效性的研究。

- 理性主义 vs 经验主义
- 可知论 vs 不可知论
- 相对主义 vs 绝对主义

### 伦理学

关于价值、道德和人生意义的研究。

- **义务论**：道德基础是义务
- **功利论**：追求最大的幸福
- **德性论**：培养美德的品质

## 💭 重要思想家

### 古代
- **柏拉图** - 理论世界论的创立者
- **亚里士多德** - 逻辑学之父

### 现代早期
- **笛卡尔** - 理性主义的奠基人
- **康德** - 批判哲学的创立者

### 现代晚期
- **尼采** - 权力意志论
- **维特根斯坦** - 语言哲学

## 🎨 不同视角的比较

```
理性主义 vs 经验主义
↓
知识来源：先天理性 vs 感官经验
↓
方法论：演绎推理 vs 归纳观察
↓
代表人物：笛卡尔、莱布尼茨 vs 洛克、休谟
```

## ❓ 常见问题

??? question "西方哲学和东方哲学有什么区别？"
    - **西方哲学**：强调理性分析，二元对立，重视逻辑论证
    - **东方哲学**：注重整体观照，和谐平衡，重视实践体悟

??? question "现代哲学为什么这么复杂？"
    随着科学的发展和社会的变化，哲学需要应对越来越复杂的问题，因此理论也变得更加复杂。
