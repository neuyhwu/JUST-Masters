# 🤖 机器学习

机器学习在优化决策中的应用越来越重要，特别是在预测和学习算法参数方面。

## 🎯 学习目标

- 理解监督和无监督学习的基本概念
- 掌握常用算法和模型
- 学会特征工程和模型评估
- 应用到实际优化问题

## 📚 核心内容

### 1. 监督学习
- 线性回归和逻辑回归
- 支持向量机（SVM）
- 决策树和随机森林
- 神经网络基础

### 2. 无监督学习
- 聚类（K-means、层次聚类）
- 降维（PCA、TSNE）
- 异常检测

### 3. 深度学习基础
- 神经网络结构
- 反向传播和优化
- CNN、RNN基础
