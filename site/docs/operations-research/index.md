# 📘 运筹学

运筹学（Operations Research）是应用科学的一个分支，主要研究如何在复杂条件下做出最优决策。本模块深入讲解运筹学的核心方法和应用。

## 🎯 学习目标

- 理解运筹学的基本概念和方法论
- 掌握线性规划、整数规划等经典模型
- 学会建立和求解实际优化问题
- 了解运筹学在运输、生产等领域的应用

## 📚 核心知识体系

### 1. 线性规划 (Linear Programming, LP)
- **模型基础**：目标函数、约束条件、可行域
- **求解方法**：单纯形法、内点法
- **灵敏度分析**：参数变化对最优解的影响
- **应用案例**：生产计划、资源配置、运输问题

### 2. 整数规划 (Integer Programming, IP)
- **模型分类**：纯整数规划、混合整数规划、0-1规划
- **建模技巧**：如何将实际问题转化为IP模型
- **求解策略**：分枝定界、切割平面、列生成
- **高级应用**：TSP、VRP、设施选址等NP困难问题

### 3. 网络流问题 (Network Flow)
- **基本概念**：图、节点、边、流量限制
- **经典问题**：最短路、最大流、最小成本流
- **应用领域**：交通网络、通信网络、供应链

## 💡 关键概念

!!! note "最优性条件"
    在运筹学中，最优解需要满足一定的数学条件。最常用的是KKT条件（Karush-Kuhn-Tucker条件），它是一阶必要条件。

!!! warning "模型假设"
    构建运筹学模型时，需要清晰阐明假设条件。不同的假设可能导致完全不同的模型和求解策略。
