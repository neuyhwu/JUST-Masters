# 🔧 Gurobi优化求解器

Gurobi是业界领先的商业优化求解器，高效求解LP、MIP、QP等问题。

## 🎯 学习目标

- 掌握Gurobi安装和配置
- 学会Python API的使用
- 构建复杂的优化模型
- 调参和性能优化

- 下载和安装Gurobi
- 许可证管理（学术许可证）
- 环境变量配置

### 2. Python接口
- Model、Var、Constraint对象
- 目标函数设置
- 参数配置

### 3. 高级功能
- 多目标优化
- 参数调优
- 自定义回调函数

### 4. 性能优化
- 模型预处理
- 启发式求解
- 并行计算
