# 🐍 Python编程

Python是科学计算和优化领域的首选语言，拥有丰富的库和强大的社区支持。

## 🎯 学习路径

- **基础语法**：Python核心特性和常用模式
- **科学计算库**：NumPy、SciPy、Pandas等
- **数据处理**：数据清洗、可视化、分析

## 📚 核心内容

### 科学计算栈
- NumPy：多维数组和矩阵运算
- SciPy：优化、统计、信号处理等
- Pandas：数据处理和分析
- Matplotlib/Seaborn：数据可视化
- Scikit-learn：机器学习库

### 优化工具集
- Gurobi Python接口：商业优化求解器
- PuLP/Pyomo：建模语言
- Scipy.optimize：通用优化工具
