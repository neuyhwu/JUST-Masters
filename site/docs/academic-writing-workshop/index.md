# 📝 学术写作工坊

欢迎来到学术写作工坊！这个模块为研究生提供全面的学术写作技能训练，涵盖专业排版工具、轻量级标记语言和人工智能辅助写作等内容。无论是撰写论文、项目报告还是学术论文，这里都能找到实用指南和最佳实践。

## 🎯 学习路径

本模块按照实用性和学习难度组织，建议按以下顺序学习：

1. **Markdown语言** ⭐ — 轻量级标记语言，适合快速记笔记和撰写通用文档
2. **LaTeX排版** ⭐⭐ — 专业学术论文排版工具，支持复杂数学公式和学术格式
3. **大模型学术写作提示词** ⭐⭐⭐ — 利用AI工具（ChatGPT、Claude等）优化学术写作的高级技巧

## 📚 核心内容

### 📖 Markdown基础与应用
- **标记语法**：标题、列表、表格、代码块等基本元素
- **格式化技巧**：强调、引用、链接和多媒体嵌入
- **工具生态**：编辑器、转换工具、版本控制集成
- **实战案例**：学术笔记、项目文档、技术博客的写作模板

### 📐 LaTeX专业排版
- **环境搭建**：在线编辑器（Overleaf）和本地安装配置
- **文档结构**：文档类、导言区配置、主体结构
- **数学公式**：单行公式、多行公式、方程组的优雅排版
- **学术元素**：引文管理（BibTeX）、图表、参考文献、索引
- **高级功能**：自定义命令、宏包选择、PDF优化

### 🤖 大模型辅助学术写作
- **提示词工程基础**：有效指令设计、上下文塑造、角色定位
- **写作场景**：论文大纲生成、段落扩展、语言优化、论证论证
- **工具集成**：与ChatGPT、Claude、Copilot等LLM的协作工作流
- **质量控制**：AI生成内容的审核、改进和学术诚信
- **最佳实践**：避免过度依赖、保持学术声音、创意与AI平衡

## 💡 学习建议

!!! note "初学者建议"
    如果你是第一次接触学术写作工具，建议先从**Markdown**开始，快速掌握基本标记语法，获得成就感。再逐步深入**LaTeX**的世界，最后学习如何**利用AI工具**辅助写作。

!!! tip "实践导向"
    每个章节都包含实际代码示例和完整项目案例。强烈建议边学边练，即时编写、编译和验证你的代码。

!!! warning "学术诚信"
    使用AI辅助写作时，务必：
    - 清楚标注AI生成的内容
    - 独立审核和修改AI输出
    - 遵循学校的AI使用政策
    - 保持学术写作的原创性和个人风格

!!! danger "常见陷阱"
    - **Markdown误用**：混淆不同方言（GitHub Flavor、Common Mark等）
    - **LaTeX过度**：为简单文档选择LaTeX导致事倍功半
    - **AI过度依赖**：完全依赖AI生成论文违反学术道德

## 🎓 应用场景

### 场景1：快速记笔记与知识整理
**工具**：Markdown + 本地编辑器（VSCode、Typora）或在线工具（Notion）  
**优势**：轻量级、版本控制友好、易于发布成网站

### 场景2：撰写学位论文
**工具**：LaTeX + Overleaf + BibTeX参考文献管理  
**优势**：专业排版、精准控制格式、适合复杂数学和学术要求

### 场景3：准备学术报告与演讲稿
**工具**：LaTeX Beamer 或 Markdown + Reveal.js  
**优势**：一致的视觉风格、易于维护、支持动画和交互

### 场景4：投稿与协作写作
**工具**：LaTeX + Git + Overleaf（支持实时协作）+ AI助手  
**优势**：版本追踪、多人协作、AI辅助检查和改进
