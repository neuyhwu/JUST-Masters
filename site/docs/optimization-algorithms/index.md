# ⚙️ 优化算法

优化算法是解决复杂优化问题的核心工具。本模块系统讲解从启发式到元启发式的算法谱系。

## 🎯 学习目标

- 理解启发式和元启发式算法的区别
- 掌握主流算法（遗传算法、粒子群、模拟退火等）
- 学会算法设计和参数调试
- 应用到实际优化问题

## 📚 算法分类

### 1. 启发式算法 (Heuristic Algorithms)
- 贪心算法
- 局部搜索
- 构造性启发式算法

### 2. 元启发式算法 (Metaheuristic Algorithms)
- 遗传算法 (GA)
- 粒子群算法 (PSO)
- 模拟退火 (SA)
- 禁忌搜索 (TS)
- 蚁群算法 (ACO)
- 差分进化 (DE)

### 3. 高级优化方法
- 分枝定界
- 切割平面
- 列生成
- 拉格朗日松弛
