# ⚙️ 优化算法 - 高级优化方法

本章讨论精确求解复杂优化问题的高级算法和技巧。

## 🎯 核心算法

### 1. 分枝定界 (Branch and Bound)
- 递归分割问题空间
- 基于松弛问题的界
- 用于MIP求解

### 2. 切割平面 (Cutting Plane)
- 通过添加有效不等式增强松弛
- 每次切割缩小可行域
- 与分枝定界结合使用

### 3. 列生成 (Column Generation)
- 针对变量众多的问题
- 主问题和子问题迭代
- 应用：运输问题、时间表制定

### 4. 拉格朗日松弛 (Lagrangian Relaxation)
- 将困难约束纳入目标函数
- 获得下界或上界
- 指导启发式算法
