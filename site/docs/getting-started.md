# 🚀 快速开始指南

欢迎！这个指南将帮助你了解如何使用这个知识库。

## 📖 基本操作

### 浏览内容

- **顶部导航**：使用顶部的标签来切换不同的主要分类
- **侧边栏**：左侧侧边栏显示当前分类的子页面
- **面包屑导航**：页面顶部显示当前位置的路径

### 搜索功能

点击页面顶部的搜索图标或按下 `Ctrl+K` (Windows) / `Cmd+K` (Mac)：

- 输入关键词进行全文搜索
- 支持中文和英文
- 搜索结果会实时显示匹配的页面

### 主题切换

在页面右上角找到主题切换按钮：

- ☀️ **浅色模式**：适合日间使用
- 🌙 **深色模式**：适合夜间使用

## 🎨 Markdown 语法支持

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本格式

- **粗体**：`**文本**`
- *斜体*：`*文本*`
- ~~删除线~~：`~~文本~~`
- `代码`：`` `文本` ``

### 列表

#### 无序列表
- 项目1
- 项目2
  - 子项目2.1
  - 子项目2.2

#### 有序列表
1. 第一步
2. 第二步
3. 第三步

### 链接和图片

```markdown
[链接文本](https://example.com)
![图片描述](image.png)
```

## 🔗 页面导航

### 面包屑

在每个页面顶部，你会看到当前位置的路径，点击可快速返回。

### 目录

右侧会显示当前页面的目录，可快速跳转到特定部分。
