# 📋 关于本站

## 网站简介

**JUST-Masters 知识库**是一个个性化的学习资源汇集平台，旨在组织和分享跨学科的知识。

## 👤 关于作者

- **姓名**：JUST-Masters
- **学校**：JUST
- **研究方向**：交通运输管理、人机协同生产
- **兴趣领域**：徒步爬山、看书、运筹优化

## 🎯 网站目标

1. **知识整理**：系统地组织学习笔记和研究成果
2. **资源分享**：与他人分享有价值的内容和见解
3. **学习平台**：为学生和研究者提供参考资源
4. **个人品牌**：展示学术兴趣和思想深度

### Markdown 扩展

网站支持以下Markdown扩展：

- ✅ **admonition** - 提示框
- ✅ **pymdownx.details** - 可展开内容
- ✅ **pymdownx.superfences** - 高级代码块
- ✅ **pymdownx.emoji** - 表情符号
- ✅ **pymdownx.arithmatex** - 数学公式
- ✅ **pymdownx.tabbed** - 标签页
