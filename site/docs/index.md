# 🎓 运筹优化研究生知识库

欢迎来到 **JUST-Masters 运筹优化知识库**！这是一个面向研究生的综合学习平台，汇集了运筹学、优化算法、编程技术和机器学习的系统化资源。

## ✨ 知识库特色

- 📚 **系统化课程**：覆盖运筹学、优化算法、编程、Gurobi求解器和机器学习
- 🎯 **研究生导向**：针对高等教育和科研需求设计
- 💻 **实践代码示例**：Python、Java等语言的完整代码示例
- 🔍 **全文搜索**：快速查找相关概念和算法
- 🌓 **浅色/深色主题**：根据偏好切换显示模式  
- 📱 **响应式设计**：支持桌面和移动设备

## 🎯 推荐学习路径

### 初级研究生（第一学期）

2. [Python编程](python-programming/index.md) - 掌握工具链
3. [Python编程 - 科学计算](python-programming/scientific-computing.md) - 学习NumPy/SciPy

### 中级研究生（第二学期）

### 高级研究生（第三-四学期）
